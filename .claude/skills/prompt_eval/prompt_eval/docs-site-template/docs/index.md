# Prompt Eval Reports

Welcome. This site collects every prompt-evaluation run produced by the
`/prompt_eval` Claude Code skill.

Each run lives in the **Runs** sidebar. Within a run you'll find:

- **Summary** — models used, dataset size, average score per version
- **Comparison** — side-by-side score matrix and prompt diffs (when 2+ versions exist)
- **v1, v2, …** — full prompt text, per-case results, raw outputs

Use the search box (top-right) to find a scenario, prompt fragment, or reasoning string across all runs.
